3. Approach 2 — A2A JSON-RPC (structured tasks)

Use the A2A HTTP endpoint when you need structured output, streaming, or want to track task state. All financial skills run through this path.

Step 1 — SendMessage

POST http://10.0.0.143:9001/
Content-Type: application/json

{
  "jsonrpc": "2.0",
  "method": "SendMessage",
  "id": "my-task-1",
  "params": {
    "message": {
      "role": "ROLE_USER",
      "parts": [{ "text": "Summarize the January statement" }]
    },
    "skill": "summarize_statement",
    "context": {
      "statementPath": "/home/adamsl/rol_finances/readable_documents/bank_statements/january/chase_jan.txt"
    }
  }
}

Response — save the result.id as your task ID:

{
  "jsonrpc": "2.0",
  "id": "my-task-1",
  "result": {
    "id": "<task-uuid>",          // ← save this
    "status": { "state": "SUBMITTED" }
  }
}

Step 2 — Poll GetTask until COMPLETED

POST http://10.0.0.143:9001/
{
  "jsonrpc": "2.0",
  "method": "GetTask",
  "id": "poll-1",
  "params": { "id": "<task-uuid>" }
}

// States: SUBMITTED → WORKING → COMPLETED (or FAILED / CANCELED)
// result.result contains Jeri's answer when COMPLETED

Task state flow

SUBMITTED → WORKING → COMPLETED  or  FAILED

Streaming (SSE) — SendStreamingMessage

POST http://10.0.0.143:9001/
{
  "jsonrpc": "2.0",
  "method": "SendStreamingMessage",
  "id": "stream-1",
  "params": { /* same params as SendMessage */ }
}
// Response is text/event-stream — each data: frame is JSON
// data: {"type":"chunk","taskId":"...","content":"text..."}
// data: {"type":"done","taskId":"...","status":"COMPLETED"}

7. When to Use Which Approach

SituationUse
Quick question about a statement or the SDKLetta native (send_letta_message)
Need structured JSON output (transactions, anomalies)A2A SendMessage + GetTask poll
Want to see output as it streams token by tokenA2A SendStreamingMessage (SSE)

Cancel an in-flight taskA2A CancelTask

8. Starting & Stopping Jeri's A2A Server

The Letta agent is always on (it's registered on the server). The A2A HTTP server must be started separately on Mom's Machine:

# Start (from /home/adamsl/claude-code-sdk-ts)
cd agents/jeri && bash start_jeri_agent.sh

# Stop
bash stop_jeri_agent.sh

# Development mode (hot reload)
npx tsx /home/adamsl/claude-code-sdk-ts/agents/jeri/index.ts
Note: Jeri's A2A server at http://10.0.0.143:9001 must be running for all five financial skills to work. If the health check fails, start it with start_jeri_agent.sh before delegating tasks.
