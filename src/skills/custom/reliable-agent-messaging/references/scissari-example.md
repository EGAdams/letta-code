# Quinn ↔ Scissari Example

## What went wrong

Scissari had previously received ROL Finances handoff material, but later replied as if she did not have the handoff context in-session.

This showed that:
- memory existing is not the same as memory surfacing reliably in the next reply
- agent-to-agent messages need stronger verification patterns

## Better pattern for this case

Instead of asking only whether Scissari was up to speed, ask her to:
- summarize the current ROL Finances state
- mention the active pipeline entrypoint
- mention the `vendor_key` / `id_light` ordering fix
- list the next 3 planning tasks

That produces a much more trustworthy answer.
