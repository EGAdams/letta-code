# SPA Plan (reports dashboard)

## Goal
Build a simple HTML/CSS/JS SPA under `public_html/reports` matching `largo_spa` look/feel.

## Steps
1. Inspect existing `largo_spa` layout/styles and invoice-tab behavior.
2. Identify target reports directory/files for 2025 month-by-month links.
3. Build `reports` SPA with left-side month tabs (Jan-Dec 2025).
4. Fill January with completed work summary + current links.
5. Add placeholder summaries + easy-to-edit link structure for other months.
6. Validate locally and deploy/sync to `public_html/reports`.
7. Document files created/changed.
